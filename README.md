**SharapSharapBot**

Telegram-бот @SharapSharapBot на базе языковой модели gemma4:31b-cloud, запущенной через Ollama, с автоматической генерацией постов.

**Что умеет:**

* Отвечает на сообщения в личных чатах и группах через LLM (Ollama)
* Автоматически ищет в интернете через Tavily API при информационных вопросах
* Сохраняет переписки и хранит память диалога
* Можно вызывать в комментариях под постом, если добавить бота в сообщество. Бот вникает в контекст постов и других комментариев.
* Генерирует посты для Telegram-канала через агентный пайплайн: несколько кандидатов генерируются параллельно, отдельная LLM-модель оценивает и выбирает лучший
* Обучается на фидбэке: посты оцениваются вручную, оценки сохраняются и используются как few-shot примеры для следующих генераций
* Каждый день автоматически отправляет утреннее приветствие с фото и аудио, используя случайную ролевую маску
* Периодически сам постит анекдоты в канал, есть команды для ручного вызова
* Обрабатывает изображения, стикеры и видео-стикеры (конвертация через ffmpeg)
* Ставит случайные emoji-реакции на сообщения
* Логирует все действия через Grafana Loki, метрики — через Prometheus. Ведется подсчёт экономики токенов.
* Ведёт учёт расхода токенов LLM для мониторинга затрат. 

**Требования:**

* Python 3.10+
* Ollama (https://ollama.com) — должна быть запущена локально или доступна как облачный эндпоинт
* PostgreSQL база данных (для хранения истории чатов и фидбэка)
* Tavily API ключ (опционально, для веб-поиска)

**Структура проекта:**

* sharapbot.py — точка входа, обработчики Telegram-команд и сообщений
* llm.py — работа с моделью, контекст диалога, поиск
* agent.py — генерация нескольких кандидатов постов и выбор лучшего
* feedback.py — сохранение и загрузка фидбэка для обучения агента
* quote_prompts.py — построение промптов для генерации постов
* padique_examples.py.example — пример структуры файла с образцами постов
* morning_prompts.py.example — пример структуры файла с ролями для утренних приветствий 
* requirements.txt — зависимости Python
* tokens.env.example — пример структуры файла с токенами

**Примечание:**
