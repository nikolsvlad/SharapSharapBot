# SharapSharapBot

Telegram-бот @SharapSharapBot на базе облачной языковой модели DeepSeek, запущенной через Ollama.

## Что умеет

- Отвечает на сообщения через облачную модель deepseek-v3.2 (Ollama)
- Автоматически ищет в интернете через Tavily при информационных вопросах
- Помнит контекст диалога в личных чатах и группах
- Сохраняет историю переписки в Supabase
- Добавляет случайные реакции на сообщения
- Каждый день автоматически отправляет утреннее приветствие с фото и аудио
- Периодически сам скидывает анекдот в чат
- Можно вручную вызывать анекдоты через команды
- Логирует все действия через Grafana Loki

## Требования

- Python 3.10+
- Ollama (https://ollama.com) — должна быть запущена локально
- Модель deepseek-v3.2:cloud загружается автоматически при первом запуске

## Структура проекта

- sharapbot.py — основной файл бота
- llm.py — работа с моделью, поиском и контекстом
- hilarious_stuff/ — медиафайлы для команд
- rooster/ — картинки и аудио для утреннего приветствия 
- requirements.txt — зависимости Python
- tokens.env — токены
